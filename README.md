# 🔮 CryptoSeer — ML Crypto Price Predictor

A machine learning web app that predicts cryptocurrency prices using Random Forest and Gradient Boosting models.

## 🚀 Features
- 📈 Interactive candlestick price charts (2014–2026)
- 🧮 Technical indicators: MA, RSI, Volatility
- 🤖 ML models: Random Forest, Gradient Boosting, Linear Regression
- 📊 Multi-coin comparison with correlation heatmap
- 🔮 Tomorrow's price prediction with Bullish/Bearish signal

## 🛠️ Tech Stack
- Python · Streamlit · Scikit-learn · Pandas · Plotly

## ▶️ Run Locally
```bash
pip install streamlit pandas scikit-learn plotly
streamlit run crypto_app.py
```

## 📂 Dataset
[Top 50 Cryptocurrency Data 2014–2026](https://www.kaggle.com/datasets/kaushalnandania/crypto-data-2014-2026) from Kaggle
